# bot_discord
Discord bot to train JS and discord API !

## Execute project :
#### Install packages
    npm install

### Make a "config.json" on directory base :
```json
{
    "TOKEN":"HereGoesYourDiscordBotToken",
    "bdd_password": "The pasword of the database"
}
```
#### Execute main file
    node .
