# Linkstart-bot
Discord bot to train JS and discord API !

## Execute project :
#### Install packages
    npm install

### Make a file "config.json" in a folder called "ressources" with these informaiton :
```json
{
    "clientId": "your_application_id",
    "guildId": "your_discord_server_id",
    "token": "he_token_of_your_bot"
}
```
#### Execute main file
    node .
